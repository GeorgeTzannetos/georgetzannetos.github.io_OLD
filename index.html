Lane Detection using Synthetic Dataset

Augmented Reality Data Generation for Deep Learning Based Lane Detection

Click to read more

Project Description

×
In this project, an automatic process to generate a labeled dataset for lane detection was developed. Then, this dataset was used to train a model to perform lane detection based on the direct perception approach of autonomous driving cars. To generate the dataset the following pipeline was developed:
  • Depth Completion, unguided approach was used, exploiting only the depth info from LIDAR data.
  • Road Segmentation, a deterministic region growing method was developed to compare neighbouring points and detection and extract the free space.
  • Image Augmentation, a heuristic method was programmed in Blender using the extracted road above to augment real roads with virtual lanes in Blender and then the final image was rendered along with the corresponding label.
Finally, this dataset was used to train a network for lane detection. The performance was equivalent with networks trained on real datasets.

Robotics & Game Theory Project

A non cooperative game between an unmanned aerial vehicle and an unmanned ground vehicle.

Click to read more

Project Description

×
In this project a non cooperative game between a UAV and a UGV was investigated and examined in simulations and experiments.

The goal was to research the techniques that two independent agents use in order to achieve their objectives. For the aerial to minimize the distance and for the ground vehicle to increase the relative distance.

First part is to analyze the dynamic models of the two vehicles and then identify the unknown parameters that best describe the performance of the models. This one achieved with grey-model system identification.

For controling the vehicles a Model Predictive Control was used with fixed horizon, where the obejctive function is a MinMax function, where each agent estimates the next dynamic state of the opponent using the previous states. The estimation is being corrected by sensor measurements.

Main sensors used were camera(position and velocity using optical flow), IMU and altimeter and odometry sensors. Extended Kalman Filter was used to fuse the measurements for better state estimation and for closed-loop feedback.

The solution was examined both in simulation using Gazebo and ROS and also in real experiments and can be seen in the next videos.

Real experiments of short duration for testing the simulation results.
